Have you ever had a moment in your life where you wanted to find new music, but you didn't know where to look? We've created the tool just for these moments. This application allows a user to input a song and artist and then receive 10 song recommendations with similar audio features.

Learn About Euclidean Distance

Using Euclidean Distance to Recommend Songs

Behind the scenes, a dataset of 500k songs is filtered down to only include songs that are approximately the same release year as the user's input. Then 100 of these songs are randomly sampled, and the euclidean distance is computed for each of these 100 songs to determine which 10 songs are closest to the input song in a multi-dimensional space. The 10 songs with the smallest euclidean distance between their audio features and the input song's audio features will be suggested. The features of each song are numerical and they quantify metrics like tempo, danceability, liveliness, etc.
